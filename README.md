# Opportunity Radar

Opportunity Radar is a platform that helps students discover global opportunities such as **hackathons, scholarships, internships, fellowships, and competitions** in one place.

The app collects opportunities from across the web, verifies them, and displays them in a clean feed so users can easily find and apply before deadlines.

## Key Features

* Curated list of global opportunities
* Opportunities sorted by **deadline urgency**
* **Dark mode toggle**
* **Email notifications** for new opportunities
* **Automated scraping** to discover opportunities across the web
* **Pagination** to browse more opportunities

The development server will start, and the app will be available locally.

Built to help students **never miss life-changing opportunities.**
